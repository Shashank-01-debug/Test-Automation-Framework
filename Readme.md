
# Test Automation Framework (Java + TestNG)

This repository contains a Java-based Test Automation Framework designed for scalable, maintainable, and efficient UI testing. The framework follows best practices such as modular design, data-driven testing, and reusable utilities.
It is built using modern tools and libraries to support cross-browser execution, headless execution, and detailed reporting.

##  Prerequisites

Ensure you have the following installed:

Java 11
Maven
Browser drivers (ChromeDriver, GeckoDriver, etc.)
IDE (IntelliJ / Eclipse - optional)

 Features
- Cross-browser testing (via parameterization)
- Headless execution support for faster runs
- Data-driven testing using CSV & Excel
- Reusable Page Object Model (POM) design
- Thread-safe reporting with Extent Reports
- Logging with Log4j for debugging & traceability
- Fake data generation using Faker library
- Maven-based execution for CI/CD compatibility

Technologies used
-Language: Java 11
-Build Tool: Maven
-Test Framework: TestNG
-Automation: Selenium WebDriver
-Data Handling:
-OpenCSV (CSV-based data-driven testing)
-Apache POI (Excel-based data-driven testing)
-Fake Data Generation: Java Faker
-Reporting: Extent Reports
-Logging: Log4j

## Installation

git clone https://github.com/Shashank-01-debug/Test-Automation-Framework.git
cd Test-Automation-Framework

run using maven 

mvn test -DisLambdaTest= false -Dbrowser=chorme -DisHeadLess = true -x
